PORTFÓLIO

Automação Industrial

I.A

A disciplina de Automação Industrial tem como objetivo fornecer aos estudantes uma compreensão abrangente dos princípios, tecnologias e aplicações da automação nos processos industriais.

Desenvolvimento Web

W.D

A disciplina de Desenvolvimento Web visa proporcionar aos estudantes uma compreensão sólida dos princípios, ferramentas e técnicas necessários para criar e manter sites e aplicações web.

Banco de Dados

D.B

A disciplina de Banco de Dados tem como objetivo fornecer aos estudantes um entendimento abrangente sobre os princípios e práticas de design, implementação e gerenciamento de bancos de dados. O curso aborda tanto os sistemas de gerenciamento de bancos de dados relacionais (SQL) quanto os bancos de dados NoSQL, com ênfase especial no MongoDB.

Linguagem de Programação

L.P

A disciplina de Linguagem de Programação com foco em Java visa proporcionar aos estudantes uma compreensão aprofundada dos conceitos fundamentais da programação orientada a objetos (POO) e habilidades práticas para desenvolver aplicações robustas e eficientes utilizando a linguagem Java.

Ciência de Dados

D.SCI

A disciplina de Ciência de Dados visa fornecer aos estudantes uma compreensão abrangente dos conceitos, métodos e ferramentas utilizados na análise e interpretação de grandes volumes de dados.
