Linguagem de programação

A linguagem de programação é um instrumento fundamental na era digital, permitindo aos desenvolvedores criar, implementar e manter software eficiente e escalável. Mais do que apenas um conjunto de regras sintáticas e semânticas, uma linguagem de programação representa uma ferramenta poderosa para materializar conceitos complexos em soluções práticas. Desde aplicações de desktop até sistemas distribuídos na nuvem.

Compiladores & Interpretadores:

O compilador gera um arquivo executável que pode ser executado diretamente no sistema operacional. Por outro lado, um interpretador traduz o código fonte linha por linha durante a execução.

Estruturas de repetição (exemplo while):

estrutura de repetição é uma lógica que repete as ações de um mesmo bloco de código por um número de vezes determinado a partir do tipo de estrutura de repetição.




POO (Programação Orientada Objeto):

utiliza objetos como elementos fundamentais na construção da solução. Um objeto é uma abstração de algum fato ou ente do mundo real, com atributos que representam as suas caraterísticas ou propriedades, e métodos que emulam o seu comportamento ou atividade.
